# Architecture Decision Records

One file per major technical decision, numbered `NNNN-short-title.md`. Required whenever we
deviate from a §4 default in `SPLATCAST_PLAN.md` (plan §7).

Template:

```markdown
# NNNN — Title

- **Date:** YYYY-MM-DD
- **Status:** proposed | accepted | superseded by NNNN

## Context
What forced the decision.

## Decision
What we chose.

## Consequences
What becomes easier/harder; follow-ups.
```
